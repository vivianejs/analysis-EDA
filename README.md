# analysis-EDA

Projeto autoral para explorar dados de filmes (EDA) e treinar modelos de ML:
- Classificação de gênero
- Regressão de nota IMDb
- Regressão de faturamento (sem leakage)

> Inspirado conceitualmente em trabalhos públicos sobre análise de filmes, mas **todo o código e a estrutura são originais**.
